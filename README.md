# Tavleen Kaur Personal Website

A Linktree-style website built with React and Vite. I created it as a personal hub for my links, resume, projects, and contact information, with a design inspired by my pet birds.

## Running Locally

```bash
npm install
npm run dev
```

## JavaScript Feature

I chose to build a dark mode / light mode toggle. The site remembers the user's theme preference using localStorage, so the selected theme stays the same even after refreshing the page.

I picked this feature because it's useful, simple, and something I wanted to learn more about in React. The trickiest part was understanding how React state and localStorage work together to update and save the theme.
